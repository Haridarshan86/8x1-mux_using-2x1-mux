# 8×1 Multiplexer Using 2×1 Multiplexers (Structural Modeling)

## 📌 Overview

This project implements an **8×1 Multiplexer (MUX)** using **seven 2×1 Multiplexers** in Verilog HDL through **Structural Modeling**.

The design demonstrates hierarchical hardware design by constructing a larger multiplexer from smaller reusable 2×1 MUX modules.

---

## 🎯 Objective

To design and simulate an **8×1 Multiplexer** using only **2×1 Multiplexers** and verify its functionality using a Verilog testbench.

---


---

## 🔧 Module Description

### 2×1 Multiplexer

A 2×1 MUX selects one of two inputs based on a single select line.

### 8×1 Multiplexer

The 8×1 MUX is built using seven 2×1 MUXes arranged in three stages.
